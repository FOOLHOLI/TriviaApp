TriviaApp
=========

A Trivia App built using Swift for IOS7 and IOS8 

### What is included

   * Creating a new project.
   * Grand Central Dispatcher
   * How to fetch data from web using Swift.
   * Tweet Functionality
   * Run in Simulator
